# simple

Safe Rust bindings for simple, automatically generated.

## Installation

```toml
[dependencies]
simple = "0.1.0"
```

## Prerequisites

simple must be installed on your system.

## Usage

```rust
use simple::*;

fn main() -> Result<(), Error> {
    // Use the generated bindings
    Ok(())
}
```

## Documentation

Generate documentation with:

```bash
cargo doc --open
```

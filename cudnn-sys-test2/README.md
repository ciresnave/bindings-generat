# cudnn

Safe Rust bindings for cudnn, automatically generated.

## Installation

```toml
[dependencies]
cudnn = "0.1.0"
```

## Prerequisites

cudnn must be installed on your system.

## Usage

```rust
use cudnn::*;

fn main() -> Result<(), Error> {
    // Use the generated bindings
    Ok(())
}
```

## Documentation

Generate documentation with:

```bash
cargo doc --open
```
